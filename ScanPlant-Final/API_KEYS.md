# ⚠️ IMPORTANTE: Configure suas API Keys

Este projeto usa as seguintes APIs externas que requerem chaves:

## 1. Groq API (IA para identificação de plantas)
- Site: https://console.groq.com
- Crie uma conta gratuita
- Gere uma API Key
- Configure em `PhotoScreen.js` linha 43

## 2. Plant.id API (Identificação visual)
- Site: https://web.plant.id
- Crie uma conta
- Obtenha sua API Key
- Configure em `PhotoScreen.js` linha 41

## Como configurar:

1. Copie `.env.example` para `.env`
2. Adicione suas chaves no arquivo `.env`
3. As chaves serão carregadas automaticamente

⚠️ **NUNCA faça commit do arquivo `.env` com suas chaves reais!**
